# unnotech

## 專案執行
將該 repo 自 github clone 下來或是直接下載 zip 檔，取得之後於 terminal 執行 npm install ，以將該專案的相依套件node_modules函式庫注進該專案中。安裝完成之後，於 terminal 執行 npm run serve 建立整體環境並執行。
